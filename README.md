# HOSA Plus

Beginner-friendly React + Vite starter for a HOSA project.

## Run The App

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Then open the local URL that Vite prints, usually:

```text
http://localhost:5173/
```

## Folder Structure

```text
src/
  components/  Reusable layout and UI pieces
  data/        Easy-to-edit project information
  pages/       Route pages for the app
  styles/      Tailwind CSS entry file
```

## Edit Your Project Content

Start with `src/data/projectInfo.js`. That file controls the project name,
overview, goals, and resource links shown across the site.
